## Metadata Parsing

The application supports multiple AI image generators:

- InvokeAI
- Automatic1111
- ComfyUI
- SwarmUI
- Easy Diffusion
- Midjourney/Niji Journey
- Forge
- DALL-E
- Adobe Firefly
- DreamStudio
- Draw Things
- Flux (via various UIs)

Supported File Formats:

- Images: PNG, JPEG, WEBP
- Animation: GIF
- Video: MP4, WEBM

Metadata sources:

- PNG chunks (tEXt, iTXt, zTXt)
- JPEG EXIF/XMP
- Sidecar JSON files
- C2PA manifests
